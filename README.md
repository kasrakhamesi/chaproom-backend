# Chaproom-Backend

## Setup Guide

### 1. Setup Database config

The definition for all the configs configuration can be found in `.env`

## Get Started

Rebuild database and run in development mode

```
npm run rebuild
```

Development

```
npm run dev
```

Production

```
npm start
```
